This folder contains the manually curated study annotations from CREEDS, a resource provided by Avi Ma'ayan's lab.

The annotation files, single_gene_fixed.txt and single_drug_fixed.txt, are manually altered to remove artifacts and formatting issues. The original files can be found here:
https://maayanlab.cloud/CREEDS/download/single_gene_perturbations-v1.0.csv
https://maayanlab.cloud/CREEDS/download/single_drug_perturbations-v1.0.csv

fixed_kokd_terms.txt is a manually constructed list of terms from the single-gene file that we believe pertain to knockout and knockdown experiments.
